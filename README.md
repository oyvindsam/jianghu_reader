# jianghu_reader
This is a project i have been working on since Feb 2017. I read a lot of webnovels and wanted to make the reading experience on mobile better, fitting my needs. As this is my first step into Android development I know there is a lot of stuff to improve, and hopefully I will make time to update this project in the coming future.



DOWNLOAD: https://github.com/oyvindsam/jianghu_reader/blob/master/Additional%20files/jianghu_reader_r1.apk



Features:

-Save favorites

-Easy, clean UI with "nightmode"

-Tap to scroll 

-Load last read chapter (fab button)

-Load chapter from link (overflow menu), ps. chapter must match novel
